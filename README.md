# Dust Evolution Comparison Study

This repository contains all scripts used for initializing, running, and post-processing the simulations in the dust evolution comparison study by **Eriksson et al. (2026)**:

*Turning the knobs on dust evolution: Comparing codes, parameters and their effects on planet formation and disk observables*

---

## 1D Radial Simulations

### Codes
- DustPy
- TriPoD
- two-pop-py

### Setups
- Unperturbed viscous accretion disk
- Viscous accretion disk with a planetary gap

### Post-processing scripts
- Planetesimal formation
- Pebble accretion
- Calculation of observables

---

## 2D Radial–Vertical Simulations

### Codes
- TriPoD
- cuDisc
- mcdust

### Setup
- Power-law disk where gas evolution is omitted
