# 2D radial-vertical simulations
Scripts for setting up and running the 2D radial-vertical simulations of Eriksson et al. (2026). 

### Codes
- TriPoD (based on the TriPoD fork of Athena++: https://github.com/ThomasPfeil/athena_master_TriPoD)
- cuDisc
- mcdust
- DustPy (used for comparison between 1D/2D codes)
